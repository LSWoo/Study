# 산술연산
- 사칙연산
- 대입연산

```C++
int a = 1;
// 대입 : a라는 바구니를 할당하고 1을 넣는다.

// 사칙연산
a = a + 3; a += 3;
a = a - 3; a -= 3;
a = a * 3; a *= 3;
a = a / 3; a /= 3;
a = a % 3; a %= 3;

// 증감 연산자
a = a + 1; a++

```
# 비교연산
```C++
a == b; // a 와 b 의 값이 같다.
a != b; // a 와 b 의 값이 다르다.
a >= b; // a 가 b 보다 크거나 같다.
a <= b; // a 가 b 보다 작거나 같다.
```

# 논리연산
```C++
! // Not 0이면 1을 반환, 1이면 0을 반환
&& // And 둘다 1이면 1을 반환, 둘중 하나라도 0이라면 0을 반환
|| // Or 둘중 하나라도 1이면 1을 반환, 둘다 0이면 0을 반환
```
