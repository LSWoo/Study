# 반복문

## while
```C++
while (조건)
{
  명령문
}
```
```C++
int count = 0;
while (count < 5)
{
  cout << "Hello World" << endl;
  count++;
}
```
## do while
- while 과의 차이점은 while 은 조건을 만족하지않으면 명령문을 실행하지 않지만,  
do while 은 조건을 만족하지않더라도 무조건 명령문을 한번 실행 후 종료된다.
```C++
do
{
  명령문
} while (조건);
```
```C++
do
{
  cout << "Hello World" << endl;
} while (fale);
```
## for
```C++
for (초기식; 조건식; 제어식)
{
  명령문;
}
```
```C++
for (int i = 0; i < 5; i++)
{
  cout << "Hello World" << endl;
}
```
