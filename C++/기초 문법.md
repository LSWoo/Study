# C++ 

## 변수

### 정수
- char (byte)
- short (2byte)
- int (4byte)
- __int64 (8byte)

```Text
[타입] [이름] = [초기값];
int A = 100;
(signed) int A = 100;
unsigned int A = 100;
기본적으로 정수에는 signed 가 생략되어있다.
unsigned 를 사용하면 양수만 존재한다. 
byte 기준 최상위 비트가 1이라면 -127 인데, unsigned 을 사용하면 128이 된다.
초기값이 있으면 section .data 부분으로 이동한다.
초기값이 0이거나 초기값이 없는 변수라면 section .bss 부분으로 이동한다.
```
