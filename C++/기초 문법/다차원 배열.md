# 다차원 배열

```Text
다차원 배열은 일차원 배열로 표현이 가능하다.
포인터와는 완전 다른 개념이다.
배열의 이름이 배열의 시작 주소를 가르킨다고 포인터와 헷갈리지말자.
```

```C++
int main()
{
  int Array2D[2][5] = 
  {
    // 2차원 배열
    {1,2,3,4,5},
    {10,12,23,34,45}
  }
    int Array2D[10] = 
  {
    // 배열을 5개로 나눠 for 문으로 출력하게되면 2차원 배열과 같은 결과가 나오므로 2차원 배열은 배열을 보기 좋게 만든것이다.
    {1,2,3,4,5,10,12,23,34,45}
  }
  

  return 0;
}
```
