## 접근 지정자
- Public : 어디에서나 접근 가능한  
- Protected : 상속받은 Class 와 나 자신만 사용하능 
- Private : 나 자신 외에는 누구도 접근할수없다.
```C++
class Car()
{
public:
    void MoveHandle() {};
    void PushPedal() {};
    void OpenDoor() {};
    
    void TurnKey()
    {
      // ...
      RunEngine();
    }
private:    
    void DisassembleCar() {};
    void RunEngine() {};
    void DisassembleCar() {};
public:
    // 엔진
    // 핸들
    // 미션
    // 배선
}

int Main()
{
    Car car;
    
}
```
