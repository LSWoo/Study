# 분기문
-

```C++
int hp = 100;
int damage = 90;

hp -= damage;
bool isDead = (hp <= 0);
```
```C++
// if (isDead == true) 와 같다.
if (isDead)
  cout << "몬스터가 죽었습니다." << endl;
  
// if (isDead == false) 와 같다.
if (!isDead)
  cout << "몬스터가 반격했습니다." << endl;

```
```C++
if (isDead)
  cout << "몬스터가 죽었습니다." << endl;
else
  cout << "몬스터가 반격했습니다." << endl;
```
```C++
if (isDead)
  cout << "몬스터가 죽었습니다." << endl;
else
{
  if (hp <= 20)
      cout << "몬스터가 도망갔습니다. << endl;
  else
      cout << "몬스터가 반격했습니다." << endl;
}
```
