# 다중 포인터

```C++
void SetMessage(const char* a);

void SetMessage(const char** b);

int main()
{
	"Hello Wolrd"; // const char[] 형식 
	// msg [Hello 시작 주소]
	const char* msg = "Hello";


	// 1. [매개변수][RET][지역변수 msg(Hello 시작 주소)][매개변수 a(Hello 시작 주소)][RET][지역변수]
	SetMessage(msg);
	SetMessage(&msg);
	// 3. [매개변수][RET][지역변수 msg(Hello 시작 주소)
	cout << msg << endl;

}

void SetMessage(const char* a)  // cost char* 는 a 의 값이 아닌 a 의 주소값을 가지고있다.
{
	a = "Bye";
	// 2. [매개변수][RET][지역변수 msg(Hello 시작 주소)][매개변수 a(Bye 시작 주소)][RET][지역변수]
    // a 의 주소값을 Bye 의 시작 주소로 바꾼것이지 값 자체를 변경한것은 아니다.
	// 따라서 a 의 주소값의 값에 다가가기 위해서 다중 포인터를 사용해야한다.
}

void SetMessage(const char** b)
{
	// b 는 msg 의 주소값을 가르키고있다.
	// *b 는 Hello 의 시작 주소를 가르키고있다.
	*b = "Bye";
}
```
