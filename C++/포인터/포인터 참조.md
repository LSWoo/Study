### 참조와 포인터의 차이점
- 참조는 값 복사 방식과 보기에 차이점이 없기 때문에 주의해야한다.
```C++
Struct Statinfo
{
  int hp;
  int attack;
  int defence;
}

void PrintByRef(Statinfo& info)
{
   cout << info.hp << endl;
   cout << info.attack << endl;
   cout << info.defence << endl;
}
```
