** bit 표현 단위**
- 8bit = 1byte
- 16bit = 2byte = 1word
- 32bit = 4byte = 2word = dword (double-word)
- 64bit = 8byte = 4word = qword (quad-word)  

**레지스터의 종류**
기본적으로 가장 많이 사용되는 레지스터 4가지이다.  
- RAX : 누산기 레지스터
- RBX : 베이스 레지스터
- RCX : 카운터 레지스터
- RDX : 데이터 레지스터  
레지스터는 사용 환경에 따라 앞자리가 바뀌는데 R 로 시작하는 이유는 프로세스가 64비트 환경이라 그렇다.  
32비트 환경에서는 E 로 시작하고 사용 환경에따라 앞자리가 바뀐다. 
- RAX = 64bit
- EAX = 32bit
- AX = 16bit
- AH, AL = 8bit
