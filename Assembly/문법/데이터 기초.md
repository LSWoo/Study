컴퓨터가 데이터를 어떻게 저장하는가?  정기적인 신호 유무를 이용해서 무언가를 판단한다. 0 = false, 1 = true 이런 기초적인 단위 자체를 bit 라고한다,.

**Bit**
- 0 과 1 두 가지 값만 가질 수 있는 측정 단위 
**Byte**
- 8개의 Bit 로 구성된 데이터의 양을 나타내는 단위
```C++
   0     1      0      1       0      1      1      0  
[1Bit] [1Bit] [1Bit] [1Bit] [1Bit] [1Bit] [1Bit] [1Bit]  
```
