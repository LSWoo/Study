# section
## section.data
- 초기화 된 데이터  

**변수의 선언**  
[변수이름] [크기] [초기값]  
ex) a db 0x11  

**변수의 크기**
- db(define byte) 1byte
- dw(define word) 2byte
- dd(define double) 4byte
- dq(define quad) 8byte

## section.bss
- 초기화 되지 않은 데이터  

**변수의 선언**  
[변수이름] [크기] [개수]  
ex) e resb 10  

**변수의 크기**
- resb 1byte
- resw 2byte
- resd 4byte
- resq 8byte

**data 와 bss 의 차이점**
- data 는 File 에 초기값을 모두 저장하고있지만 bss 는 초기값이 모두 0인상태로 시작해 File 의 크기가 줄어드는 장점이있다.

**메모리와 레지스터끼리 값 주고 받기
```Assembly
mov rax, a 
; a라는 변수의 값이 아닌 주소값을 rax 에 복사하는것이다.
mov rax, [a] 
; a라는 변수의 주소값이 값을 rax 에 복사하는것이다.

section.data
a db 0x11
b dw 0x2222
c dd 0x33333333
d dq 0x4444444444444444

section.bss
e resb 10
```
