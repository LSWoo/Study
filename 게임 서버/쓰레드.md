
### 쓰레드 생성
```C#
System.Threading; 

Class Program()
{
   static void main(string[] args)
    {
       Thread t = new Thread(실행할 함수);
     }
}
```
### 쓰레드 사용 
```C#
Class Program()
{
   static void TestThread()
   {
       Console.WriteLine("Hello Thread");
    }
   static void main(string[] args)
    {
       Thread t = new Thread(TestThread);
       t.Start();
     }
} 
```
